# Bluetooth-Car
Tiny car controlled using bluetooth on any android device.

## Components

* Arduino Nano
* Two DC gear motors
* Two robot wheels
* Chassis
* HC-05 Bluetooth Module
* Four (4) AA batteries and battery holder
* L293D/L298N Motor Driver IC

## Connections

HC-05 | Arduino
---|---
VCC | 5V
GND | GND
 Rx | Tx
Tx| Rx
